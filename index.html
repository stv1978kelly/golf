👥 Groups Feature

How it works:
① Create round with 5+ players → groups auto-shown in setup (max 4 each)
② Groups saved to Firebase with the round automatically
③ Player joins with room code → selects their name
④ App detects which group they are in automatically
⑤ A banner shows "Group X scoring: Names..."
⑥ They can only edit scores for players in their group
⑦ Admin (PIN 9160) can always edit all scores

4 players or fewer: no groups, everyone can score freely.
